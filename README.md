### Getting started

Find and replace the following values:

- **DEVELOPMENT_STRIPE_PUBLIC_KEY**
- **DEVELOPMENT_STRIPE_SECRET_KEY**
- **PRODUCTION_STRIPE_PUBLIC_KEY**
- **PRODUCTION_STRIPE_SECRET_KEY**
